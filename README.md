# Aether

Aether is a combined source workspace for two SIM/eSIM management projects:

- `simadmin/`: source imported from `3899/SimAdmin`.
- `vohive-collection/`: source imported from the VoHive collection workspace.

The first merge keeps the two projects isolated on purpose. Each project keeps
its own build files, dependencies, runtime assumptions, and documentation so the
combined repository can be validated before deeper architectural integration.
